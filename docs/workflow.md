# Workflow Diagram

QGIS Digitization  
↓  
PostGIS Database  
↓  
SQL Daily KM Calculation  
↓  
ODBC / Power Query  
↓  
Excel Score Dashboard  

## Description

This workflow shows how GIS line digitization data can be automatically transformed into a daily production score report.

The main idea is to reduce manual score calculation and provide a repeatable reporting structure for GIS production teams.
